# L02-Q012

**Difficulty:** Medium
**Domain:** Attack Surface
**Objective:** 3.2 Explain network attack vectors
**Security+ Objective Mapping:** SY0-701 Objective 3.2 - Network attack surfaces

## Scenario
A remote administration port is open on a firewall and uses a legacy protocol without encryption. Which attack surface does this primarily expose?

## Choices
A. Message-based vector
B. Supply chain vector
C. Network vector
D. Human vector

## Correct Answer
C

## Detailed Explanation
An exposed remote management port with a weak protocol is a network attack surface.

## Why the Incorrect Options Are Wrong
- A: This is not a message-based attack.
- B: Supply chain is unrelated to exposed network services.
- D: It is not primarily a human-targeted attack vector.

## Exam Tip
Network attack surface often includes exposed ports, insecure services, and weak remote access.
